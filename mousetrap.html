MouseTrap

This is a tool for solving the AnonaGram challenges. Usage:

  • Load the challenge image (once file is selected loading begins):
  • The Image will then be processed (might take a while)
  • Once the board is generated drag&drop (or tap start and exit position if on mobile) the letters within each game piece
  • If there is only one hidden letter in a given challenge part in play (so the part is solved) that letter will be automatically added to the final part
  • Pieces can be moved by either drag&drop or by clicking source and target where to move the piece
  • If the puzzle was incorrectly recognized press the button underneath to overwrite the recognized values
